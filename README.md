# Төслийн зорилго
УБТЗамын ангиудын өдөр тутам гараар бичдэг тайлан, бүртгэлийг цахимд шилжүүлэх. Сарын сүүлд гаргах ерөнхий тайланг автоматаар гаргадаг болгох. Ингэснээр мастер болон нягтлангийн ажлыг хөнгөвчлөх.
